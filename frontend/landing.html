What you get

Everything you need to
interrogate your data by voice

A complete AI pipeline — speech recognition, language model reasoning, and voice synthesis — wired together in one frictionless interface.

🎙
Voice-First Interface
Client-side Voice Activity Detection captures your speech the instant you start talking and stops automatically on silence — no button to hold.
📂
Multi-Format Upload
Drop in a CSV, Excel (.xlsx / .xls), or JSON file. The backend parses and summarises it into an AI-ready context in seconds.
🤖
LLM-Powered Reasoning
Qwen 2.5-7B on HuggingFace Router provides sharp, context-aware answers with full conversation memory — ask follow-ups naturally.
🔊
Spoken Responses
Answers are read back using the browser's native Speech Synthesis — zero latency, no audio round-trip, and a pause button to interrupt any time.
⚡
Whisper Transcription
Your voice is encoded as raw 16 kHz WAV in-browser and sent to whisper-large-v3-turbo via HuggingFace inference — no browser codec issues.
💬
Text Fallback
Not in a place to speak? The text input box is always available alongside the mic — same AI, same context, same conversation history.
